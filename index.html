3 passos. Zero complicação.

Sim, a extensão funciona como um painel lateral dentro do Google Flow (labs.google). Você precisa ter acesso ao Flow/Veo 3 para usar a extensão.
Ingredients é o recurso do Veo 3 que permite usar imagens de referência para manter personagens consistentes entre vídeos. A extensão ativa e seleciona as imagens automaticamente antes de cada prompt.
Use o SOS Mode! Ele permite capturar e baixar vídeos que já foram gerados manualmente. Role a página, capture os vídeos visíveis e baixe todos de uma vez.
A extensão é exclusiva para Google Chrome (e navegadores baseados em Chromium como Edge, Brave, etc). Requer a versão mais recente do Chrome para melhor compatibilidade.
